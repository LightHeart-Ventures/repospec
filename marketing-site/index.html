Why Repospec?

📊 Navigation-First

Describe your repository structure in a single, agent-optimized JSON file. No boilerplate, no magic.

🤖 AI-Ready

Built for LLM agents to understand codebases faster. Better context, smarter decisions.

⚡ Lightweight

A simple .repospec.json file. No dependencies, no compilation, just clarity.

🔍 Discoverable

Agents check for .repospec.json first. Immediate understanding of your codebase structure.

Quick Start

{
  "name": "my-project",
  "description": "A brief description of your project",
  "modules": {
    "api": {
      "path": "src/api",
      "description": "REST API handlers"
    },
    "core": {
      "path": "src/core",
      "description": "Business logic"
    }
  },
  "entrypoints": [
    "src/main.rs",
    "src/cli.rs"
  ]
}
